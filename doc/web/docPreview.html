Headless components

Headless components of EASy-Producer releases starting with version 1.2.0 are available from the SSE Maven repository. For using the repository: 
<repositories>    
  <repository>
    <id>sse</id>
    <name>sse</name>
    <url>https://projects.sse.uni-hildesheim.de/qm/maven/</url>
    <layout>default</layout>
  </repository>
</repositories>
Example dependency: 
<dependency>
   <groupId> net.ssehub.easy.runtime</groupId>
   <artifactId>EASy</artifactId>
   <version>1.3.10-SNAPSHOT</version>
</dependency>
For more details please refer to the developers documentation.
